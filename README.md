## 关于这款游戏

《勇者棋》原为 **4399 小游戏平台** 上的一款经典 Flash 游戏。在那个 Flash 盛行的年代，它陪伴了许多玩家的课余时光。

由于 Adobe Flash 已于 2020 年底停止支持，现代浏览器已无法运行原版游戏。出于对这款游戏的热爱，也为了能让更多玩家重温这份回忆，本项目将原版游戏完整移植到了 HTML5 平台，使其可以在任何现代浏览器中直接运行。

**原版**: 4399 小游戏 - 勇者棋 (Flash)  
**移植版**: 勇者棋 HTML5 (CreateJS)

## 游戏简介

你扮演一名刚刚就职的勇者，在新年的第一天踏上了打倒魔王的冒险旅程。游戏采用类似大富翁的棋盘机制：

- 投骰子决定移动步数
- 停留在不同格子上触发各种事件
- 在城镇购买装备、锻造升级
- 在副本中战斗获取材料
- 招募随从、寻找遗物、完成任务
- 最终挑战魔王，拯救世界

每走完一圈（12格）切换昼夜，每经过起始城镇自动保存。游戏包含丰富的随机事件和多结局系统。

## 操作方法

| 按键 | 功能 |
|------|------|
| **鼠标左键** | 点击骰子投骰、选择选项、与NPC互动 |
| **P键** | 开启/关闭自动游戏AI |
| **F12** | 浏览器开发者工具 |

**游戏内操作**：点击聊天窗口中的蓝色文字即可进行相应操作（如"前进"、"战斗"、"购买"等）。

**开发者调试（按 F12 打开控制台）**：
```js
showGM()  // 打开GM调试面板
hideGM()  // 关闭GM调试面板
```

## 新手上路

### 开局建议
1. **先熟悉操作**：第一局手动玩，了解各个格子的功能
2. **优先买装备**：城镇是购买和锻造装备的地方，好的装备是生存的基础
3. **注意耐力**：每次移动消耗耐力，耐力归零会在野外休息，可能遭遇危险
4. **存钱买地**：房屋格子可以投资买地，后期收益可观

### 各格子功能速查

| 格子 | 功能 |
|------|------|
| **城镇** | 购买装备、锻造升级、查看日志、存档 |
| **村庄** | 休息恢复、接受任务 |
| **地牢** | 进入副本战斗，获取材料和经验 |
| **竞技场** | 参加排名赛，赢取奖金和声望 |
| **公会** | 查看任务、领取奖励 |
| **马厩** | 购买和升级坐骑 |
| **温泉** | 恢复生命值 |
| **墓地** | 特殊事件，可能获得稀有物品 |
| **码头** | 投资建设，后期可坐船快速移动 |

### 核心系统

- **装备系统**：武器+护甲，5种品质，可强化+0~+9
- **随从系统**：6种职业，5个等级，战斗中提供支援
- **遗物系统**：17种遗物，提供各种被动效果
- **天赋系统**：21种天赋，影响战斗和事件
- **任务系统**：7种任务类型，完成获得奖励
- **婚姻系统**：6种伴侣，提供不同加成
- **结局系统**：10种不同结局，取决于你的冒险历程

### 进阶技巧

- **自动游戏**：按 P 键开启AI，适合挂机刷资源
- **GM面板**：按 F12 打开控制台，输入 `showGM()` 打开调试面板，可以测试各种功能、修改属性
- **锻造技巧**：材料充足时尽量锻造高品质装备
- **副本策略**：评估自身实力再进入，打不过可以逃跑

## 技术说明

- **框架**: CreateJS (HTML5 Canvas)
- **语言**: JavaScript (ES6+)
- **数据存储**: localStorage（本地存档）
- **分辨率**: 640x480（固定）
- **平台**: PC端浏览器

本项目为纯前端实现，无需后端服务器，所有游戏数据存储在浏览器本地。
