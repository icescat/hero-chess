点击"仙人岛" → 查看第1个事件，可以看到完整的概率树：

```
仙人岛到达（100%）
    ↓
抽到仙女事件（16.7%）
    ↓
    ├─ 60% → 姿势不对（什么都不给）
    └─ 40% → 触发成功
        ↓
        ├─ 60% → 武器
        └─ 40% → 护甲
            ↓
            ├─ 50% → 披外套（赠送下一套装备）
            └─ 50% → 诚实选择（装备强化+1）
```

## 🔧 如何添加新格子数据

### 1. 编辑 `wiki-events-data.js`
在 `getCellDetailData()` 函数中添加新格子：

```javascript
village: {
    title: '🏘️ 村庄',
    description: '...',
    location: '...',
    events: [
        {
            name: '住宿',
            probability: '玩家选择',
            trigger: '...',
            flow: `
                <div class="probability-tree">
                    <!-- 概率树HTML -->
                </div>
            `,
            details: [
                '...',
            ]
        }
    ]
}
```

### 2. 概率树节点样式类

- `.tree-node.root` - 根节点（蓝色）
- `.tree-node.success` - 成功（绿色）
- `.tree-node.fail` - 失败（红色）
- `.tree-node.info` - 信息（橙色）
- `.tree-node.cost` - 花费（紫色）
- `.tree-node.highlight` - 高亮（黄色）

### 3. 概率树结构模板

```html
<div class="probability-tree">
    <!-- 根节点 -->
    <div class="tree-node root">
        <div class="node-content">事件开始</div>
        <div class="node-detail">详细说明</div>
    </div>
    
    <div class="node-arrow">↓</div>
    
    <!-- 分支 -->
    <div class="tree-branches">
        <div class="tree-branch">
            <div class="probability-badge">50%</div>
            <div class="tree-node success">
                <div class="node-content">结果A</div>
                <div class="node-detail">说明</div>
            </div>
        </div>
        <div class="tree-branch">
            <div class="probability-badge">50%</div>
            <div class="tree-node fail">
                <div class="node-content">结果B</div>
                <div class="node-detail">说明</div>
            </div>
        </div>
    </div>
</div>
```
